# 当前科研实验完整总结（可直接迁移到新聊天窗口）

## 一、研究主线与当前实验目标

我当前这条实验线，核心是在 **Stable Diffusion 3.5（SD3.5）** 上构造**可控图像生成条件**，围绕“**人物性别**—**场景**—**职业/身份**—**行为**—**性别相关物体**”之间的关系，研究：

1. **在相同场景、职业、行为下，仅改变主体性别时，生成图像会如何变化**
2. **向 prompt 中注入男性相关物体 / 女性相关物体后，是否会影响图像中主体人物的性别表达**
3. **这些性别物体注入是否会影响后续模型对主体人物性别的判断**
4. **这些条件是否可能进一步影响后续的幻觉率（hallucination rate）**
5. **去掉人物信息（把人物涂黑）后，仅保留场景和物体线索，是否仍然会对后续识别或幻觉产生影响**

这条实验线本质上是在做一个**多因素可控生成 + 性别偏置/幻觉分析**的研究框架。

# 三、实验设计的演化过程

---

## 阶段 1：验证 SD3.5 本地生成能力

### 目标

* 确认本地 SD3.5 模型能否正常加载和出图
* 确认合适的参数组合
* 解决 24GB 显存下的 OOM 问题

### 关键结论

1. **普通 `pipe.to("cuda")` 会 OOM**
2. 需要使用：

   * `pipe.enable_model_cpu_offload()`
3. 推荐运行方式：

   * `PYTORCH_CUDA_ALLOC_CONF=expandable_segments:True`
   * `cpu_offload=True`
4. 1024×1024 + 40 steps 可以跑，但较慢
5. 为避免每次手动清理显存，脚本里加入了：

   * `cleanup_cuda()`
   * `torch.cuda.empty_cache()`
   * `torch.cuda.ipc_collect()`

### 目标

# 七、已得到的 gender judgment 结果与解读

你后来上传了两个统计文件：

* `gender_summary_overall.csv`
* `gender_summary_by_scene.csv`

基于这两个文件，我已经做过结论解读。

## 总体结论

**性别相关物体的注入，确实会影响主体人物性别判断。**

但影响具有明显不对称性：

* **男性物体注入**影响较小
* **女性物体注入**更容易把男性主体“拉向 female 判断”
* 女性主体整体更稳定，更少被拉偏

## 总体统计结果

四种条件分别是：

* `male_objects + male`
* `male_objects + female`
* `female_objects + male`
* `female_objects + female`

关键结果是：

* `male_objects + male`：97.78% 判为 male
* `male_objects + female`：93.78% 判为 female
* `female_objects + male`：90.00% 判为 male
* `female_objects + female`：100% 判为 female

### 解释

这说明：

* 主体人物视觉线索仍是主要因素
* 但女性物体会更明显地干扰对男性主体的 gender judgment
* 女性主体识别比男性主体更稳定

## 分场景现象

在：

* hospital ward
* soccer field
* operating room

这些场景中，`female_objects + male` 的干扰更明显。

这说明：

* 性别物体注入效应是**场景依赖的**
* 有些场景更容易受到物体线索影响
